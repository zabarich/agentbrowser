# Notice

**agentbrowser** is a TypeScript-native autonomous browser agent for Node.js.

This project is inspired by [browser-use](https://github.com/browser-use/browser-use)
by Gregor Zunic and contributors, licensed under the MIT License.

agentbrowser is an independent reimplementation using TypeScript, Playwright, and the
Anthropic SDK. It is not affiliated with, endorsed by, or a fork of browser-use. The
system prompts and DOM serialization approach are adapted from browser-use's
battle-tested designs, with modifications for the TypeScript/Node.js ecosystem.
